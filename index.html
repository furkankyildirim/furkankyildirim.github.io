WakeGuard

WakeGuard is a location tracking and alarm application that is written with React Native. WakeGuard adjusts how many meters to stay at your target location and informs you when the distance you set to your target location is reached. And now, version 2.0 is with you.

With the new version of the application, improvements were made in the back-end part. UI-UX has been updated and light-mode has been added to the app. With this update, the application no longer monitors users’ data. And with this update, the application has been made completely FREE.

You can find the new and updated open-source codes of the application on GitHub. In order to experience the application, you can download the application from Google Play Store and App Store.

aburakayaz.github.io/suchedule

Using stack:

  • Python
  • JavaScript
  • React Native
  • Flask
